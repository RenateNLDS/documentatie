# Foutmeldingen

Als een veld niet goed is ingevuld, moet voor iedereen duidelijk zijn wat er niet klopt. Hoe help je de gebruiker het beste als deze onjuiste of onvolledige informatie invult?

Belangrijk bij het aangeven van fouten:

- Controleer op het juiste moment op fouten.
- Zeg het met woorden.
- Schrijf een duidelijke foutmelding.
- Zet de foutmeldingen bij de betreffende formuliervelden.
- Gebruik niet de ingebouwde formuliervalidatie van de browser.
- Zet een samenvatting van de foutmeldingen boven het formulier.
- Geef feedback aan screenreadergebruikers.
- Screenreaderfeedback en focusmanagement na verzenden.
