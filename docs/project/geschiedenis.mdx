# Geschiedenis van NL Design System

Sinds 2018 willen diverse organisaties samenwerken aan een gebruiksvriendelijke, toegankelijke en begrijpelijk digitale dienstverlening. Het plan is samen een overheidsbreed design system te maken: NL Design System.

[In 2020](https://www.digitaleoverheid.nl/overzicht-van-alle-onderwerpen/innovatie/innovatiebudget/toekenning-innovatiebudget-2020-de-24-geselecteerde-projecten/) en [nogmaals in 2021](https://www.digitaleoverheid.nl/overzicht-van-alle-onderwerpen/innovatie/innovatiebudget/toekenning-innovatiebudget-2021/) en heeft [Innovatiebudget Digitale Overheid](https://www.digitaleoverheid.nl/overzicht-van-alle-onderwerpen/innovatie/innovatiebudget/) van het [ministerie van BZK](https://www.rijksoverheid.nl/ministeries/ministerie-van-binnenlandse-zaken-en-koninkrijksrelaties) het kernteam bij [Gebruikers Centraal](https://www.gebruikercentraal.nl) ondersteund. Het kernteam ontwikkelde een [IT-architectuur](/meedoen/als-developer/architectuur) met daarbij hulpmiddelen, en een [werkwijze](/meedoen/estafettemodel) waarmee samenwerking tussen deelnemende organisaties mogelijk is. De aanpak is een succes geworden, veel nieuwe organisaties en projecten willen nu meedoen.

## 2020

- [DUO helpt NL Design System](https://gitlab.com/nl-design-system/nl-design-system) door de code van hun componentebibliotheek UNO open source te maken, die als basis kan dienen voor NL Desgin System.
- [Eend](http://eend.nl) maakt voor gemeente Utrecht een [prototype van de Huwelijksplanner](https://huwelijk.utrecht.eend.nl/docs/site/huwelijksplanner/index.html) met de eerste versie van NL Design System.
- Innovatiebudget wordt toegekend, daarmee kan het kernteam na de zomer van 2020 drie maanden lang aan de slag.
- Het kernteam start met veel interviews met de community en doet onderzoek naar een effectieve IT-architectuur.
- De gemeente Den Haag start met de ontwikkeling van een componentenbibliotheek. "Mijn Den Haag" gaat ontwikkeld worden met componenten waar de NL Design System community van kan profiteren.

## 2021

- De eerste NL Design System heartbeat vindt plaats, waarin de community elke twee weken op dinsdag om 13.00 uur nieuwe ontwikkelingen en plannen presenteert.
- De gemeente Utrecht start ook met de ontwikkeling van een open source componentenbibliotheek.
- De gemeente Den Haag publiceert de componenten als open source.
- Open Formulieren gaat NL Design System gebruiken. Naast Den Haag en Utrecht gaan ook de volgende gemeenten profiteren van NL Design System:
