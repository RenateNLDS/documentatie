Useful Figma resources

Learning​

  • Figma design, Figma’s own go-to design resource.
  • Figma YouTube channel, full with hands-on Figma advice.
  • Figma Twitter account, information on conferences, meetups, updates, showcases, and anything Figma.
  • Figma Tokens, extensive documentation on using the Figma Tokens plugin.

Community​

  • Friends of Figma, Friends of Figma Discord channel.
  • NL Design System Figma Slack, NL Design System’s Figma Slack subchannel.

Productivity​

  • Figma Shortcuts (for MacOS and Windows)

Plugins​

Responsive design​

  • Breakpoints, allows you to design and preview components for varying viewport width/heights.

Design tokens​

  • Figma Tokens, the plugin used by the NL Design System team to handle design tokens in Figma.

Accessibility​

  • Stark, a plugin containing a vision simulator (for various forms of color vision deficiencies and blurred vision), fore-/background WCAG contrast checker, a tool to document keyboard focus order, and a tool to document page landmarks. Next to a free version offers a paid subscription plan for additional features/more tests that can be run.
  • A11y - Color Contrast Checker, a WCAG color contrast checker with tha ability to tweak colors to create contrast levels that are AA or AAA compliant.
  • A11y - Focus Orderer, another tool to document focus order of interactive elements.
  • Able – Friction free accessibility, allows you to check for WCAG compliant color contrast on elements other than frames.
  • Color Blind, generates copies of a selection of elements allowing you to preview them in eight forms of color vision deficiencies.

Handoff​

  • Measure, generates ‘red-line specs’ for height/width, margins, padding and tooltips with extended information on font and color usage, border radii, opacity, etc.
